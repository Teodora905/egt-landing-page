Този проект включва Начална страница на сайт.
Включва респонсив дизайн, менюта с подменюта /две версии - за десктоп и мобилен изглед/, отделен компонент за всяка секция. Линк за социални мрежи и swither за светла и тъмна тема.
Проектът е осъществен с React, Vite и custom css.
Изполвани са следните библиотеки:
- за иконите - react-icons
- слайдър за картинките - slick slider
- респонсив дизайн - media queries
- източник на картинките - https://picsum.photos
- JSON mocked data

Стартиране на проекта:
1. git clone https://github.com/Teodora905/egt-landing-page.git
2. cd egt-landing-page
3. npm install
4. npm run dev
5. Visit: http://localhost:5173
